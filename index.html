Light / Dark

Pequeno programinha teste para implementar o light e dark mode de uma pagina, feito com javascript puro, manipulando as variaveis do CSS contendo as cores da pagina.
Quando clicado, o toggle switch aciona um evento que chama uma funcao que puxa as variaveis do CSS, as armazena num vetor e troca por um vetor com outras cores pre-definidas, as reinsere no CSS, quando clicado de novo, o vetor que foi armazenado com as primeiras cores e' inserido de volta.
